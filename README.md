# products-ladies
 Maquetación de diseño de sitio web. El diseño fue seleccionado de un usuario de pinterest (Design Composition - 15 Things You Can't Afford NOT to Know by graphic mama) No tengo derechos sobre el diseño gráfico, estos pertenecen a graphic mama. El diseño fue tomado para poner en práctica mis habilidades de maquetación web y poder mostrarlo a empresas para postular en cargos de maquetador web o front end developer. 
 Las imagenes utilizadas en la maquetación fueron extraidas de la imagen del diseño original, algunas fueron retocadas para mejorar la vista en el navegador,
